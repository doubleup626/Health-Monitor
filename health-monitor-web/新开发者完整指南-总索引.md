## ⚠️ 新电脑部署必改项

**最重要的配置（必须修改）**:

1. **手表端服务器IP地址** ⭐⭐⭐
   - 文件: `calories/app-side/index.js`（第3行）
   - 文件: `calories/app-side/index-backup.js`（第6行）
   - 改为: 你的新电脑IP地址
   - 示例: `const SERVER_URL = "http://192.168.1.100:3000/api/health-data";`
   - **重要**: 改完后需要重新编译手表应用（`zeus preview`）

2. **数据库初始化**
   - 命令: `cd health-monitor-web/server && npm run init-db`
   - 只需执行一次

**其他可选配置**:
- DeepSeek AI API Key（如果用自己的）
- 端口号（如果被占用）

详见 **第1部分**
