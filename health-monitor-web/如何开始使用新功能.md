## 🎯 核心使用流程

### 流程A：添加新设备并绑定员工

#### 步骤1：添加设备
1. 点击 **"设备管理"**
2. 点击右上角 **"➕ 添加设备"** 按钮
3. 填写表单：
   - **设备ID**（必填）: 从手表获取，例如 `10092803`
   - 设备型号: 例如 `GT-R Series`
   - 固件版本: 例如 `3.0.1`
   - 绑定员工（可选）: 如果已有员工，可直接选择
   - 备注: 例如 `2024年采购`
4. 点击 **"确认添加"**

✅ **成功提示**: "设备添加成功"

#### 步骤2：添加员工（如果还没有）
1. 点击 **"员工管理"**
2. 点击右上角 **"➕ 添加员工"** 按钮
3. 填写表单：
   - **工号**（必填）: 例如 `W001`
   - **姓名**（必填）: 例如 `张三`
   - 年龄: 例如 `35`
   - 性别: 选择 `男` 或 `女`
   - 职位: 例如 `采掘工`
   - 班组: 例如 `A班`
   - 入职日期: 选择日期
   - 紧急联系人: 例如 `李四-妻子-13800138000`
   - 健康备注: 例如 `无高血压`
   - 照片URL（可选）: 留空即可
4. 点击 **"确认添加"**

✅ **成功提示**: "员工添加成功"

#### 步骤3：绑定设备到员工
1. 回到 **"设备管理"** 页面
2. 找到刚才添加的设备
3. 点击该设备行的 **"绑定"** 按钮
4. 在弹出的对话框中选择员工（例如 `张三 (W001) - 采掘工`）
5. 点击 **"确认绑定"**

✅ **成功提示**: "绑定成功"

**结果**:
- 设备状态变为 **"已绑定"**
- 绑定员工列显示 **"👷 W001"**
- 操作自动记录在 **"操作日志"** 中

---

### 流程B：修改设备信息

1. 在 **"设备管理"** 页面找到目标设备
2. 点击 **"编辑"** 按钮
3. 修改需要的字段：
   - 设备型号
   - 固件版本
   - 设备状态 (正常/离线/维护中)
   - 备注
4. 点击 **"确认修改"**

✅ **成功提示**: "设备更新成功"

---

### 流程C：解绑设备

1. 在 **"设备管理"** 页面找到已绑定的设备
2. 点击 **"解绑"** 按钮
3. 确认操作

✅ **成功提示**: "解绑成功"

**结果**:
- 绑定员工列显示 **"未绑定"**
- 绑定按钮重新可用

---

### 流程D：查看操作日志

1. 点击 **"操作日志"**
2. 查看所有操作记录

**支持的功能**:
- 🔍 搜索：输入操作人、目标ID或描述
- 📂 筛选：按操作类型、目标类型筛选
- 📅 日期范围：选择起止日期
- 📄 查看详情：点击 **"详情"** 按钮查看JSON详细信息
- 💾 导出日志：点击 **"导出"** 按钮导出CSV文件

**日志类型**:
- 绿色标签：添加操作（add_device, add_worker）
- 蓝色标签：编辑操作（edit_device, edit_worker）
- 红色标签：删除操作（delete_device, delete_worker）
- 黄色标签：解绑操作（unbind_device）

## 📝 常见问题

### Q1: 设备添加后显示"离线"状态？
**A**: 这是正常的。设备状态需要等手表推送数据后才会更新为"在线"。

### Q2: 如何批量添加设备/员工？
**A**: 当前版本需要逐个添加。后续版本可能会支持CSV导入功能。

### Q3: 删除的设备/员工可以恢复吗？
**A**: 当前实现是软删除（status='inactive'），数据仍在数据库中，可通过修改数据库恢复。

### Q4: 操作日志可以删除吗？
**A**: 为了审计需要，操作日志不提供删除功能。可以导出后归档。

### Q5: 如何修改员工照片？
**A**: 进入"员工管理" → 点击"编辑" → 修改"照片URL"字段 → 保存。

### Q6: 设备绑定后可以换人吗？
**A**: 可以。先"解绑"，然后重新"绑定"到其他员工即可。

### Q7: 如何查看某个员工的所有操作记录？
**A**: 进入"操作日志" → 在搜索框输入员工工号 → 筛选查看。
